# 🚀 CodeScreen

**CodeScreen** is a MERN stack coding platform where users can practice coding problems and host real-time one-to-one coding test sessions. It is designed to simulate real interview environments with live collaboration and code execution.

---

## 🧠 Features

- ✍️ Practice coding problems in an interactive editor  
- 👥 Host **one-to-one live coding test sessions**  
- ⚡ Real-time code collaboration  
- 🔐 Secure user authentication  
- 📊 Track submissions and performance  
- 🌐 Responsive and modern UI  

---

## 🛠️ Tech Stack

**Frontend**
- React JS

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB

**Real-time Communication**
- Stream

**Authentication**
- Clerk

---

## ⚙️ Installation & Setup

### Prerequisites
- Node.js
- MongoDB
- npm or yarn

### Clone the Repository
```bash
git clone https://github.com/kabir-asija/CodeScreen.git
cd CodeScreen
```

## Backend Setup
```bash
cd backend
npm install
npm start
```

## Frontend Setup
```bash
cd frontend
npm install
npm start
```

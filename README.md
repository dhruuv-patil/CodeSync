# Installation

## 1. Clone Repository

```bash
git clone https://github.com/dhruuv-patil/CodeSync.git
```

---

## 2. Install Frontend Dependencies

```bash
cd frontend
npm install
```

---

## 3. Install Backend Dependencies

```bash
cd backend
npm install
```

---

# 🔑 Environment Variables

## Frontend `.env`

```env
VITE_CLERK_PUBLISHABLE_KEY=
VITE_BACKEND_URL=
```

---

## Backend `.env`

```env
PORT=5000
MONGO_URI=
JWT_SECRET=
CLERK_SECRET_KEY=
```

---

# Run Locally

## Frontend

```bash
npm run dev
```

## Backend

```bash
npm start
```
